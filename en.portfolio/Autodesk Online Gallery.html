Autodesk Online Gallery

Exclusive sharing platform for Autodesk users

Project Introduction

Autodesk as the leader in 3D design, engineering and entertainment software, it has many products and many users, but there is no official platform to build a user community to help users share their outstanding works and promote users to learn from each other. Autodesk Gallery was thus created.

Company and Time

Autodesk | Mar 2015

Role

User Researcher, UX designer, and UI designer

Result

An official sharing community for Autodesk's 10 million users worldwide

Requirement Analysis

Business goals analysis

Understand the business goals the company is trying to achieve

As a company with an extremely long product line, Autodesk wants to provide a platform for all users who use Autodesk products to share, learn, discuss, and create a sense of belonging.

Persona

Create user personas to help teams use empathy to understand users

User journey map

By summarizing the user's entire journey through the site, we can fully analyze the problems and requirements that users will encounter at each step.

Summary of user requirements

Use user story maps to summarize and refine user requirements

Through user research, user persona analysis, user journey mapping, and communication with stakeholders, teams can come together to organize and map out user stories by:

  • Based on the user journey map, we can sort out what the user wants to achieve at each step, and from there, we can figure out what features we need to do.
  • Prioritize what is a must have and what can be done later.
  • Use the user story map to have stakeholder communication to determine which features can meet the release criteria when done.

Information architecture

How to design a simple and intuitive design architecture so that there is no learning cost for users?

After confirming the basic functions and requirements, we should first determine the website architecture to make sure that the overall architecture is simple, intuitive, and in line with user expectations, so that users can know how to use it after a glance, without increasing the learning cost and understanding burden.

Test and Product Iteration

Feature test

Ensure the usability, user satisfaction and market value of the feature before and after the official release

In order to ensure that my design meets user expectations and follows user habits, it is essential to carry out user testing, which I have done in three aspects:

  • Go to universities around Shanghai to start face-to-face focus groups and collect ideas.
  • Users are invited to the office for a 1-to-1 usability test.
  • Online one-on-one testing.

Output usability test report

In terms of usability, provide direction for the next iteration

Through the results of usability testing, summarize the usability problems and arrange them according to the severity level. If they are problems that need to be fixed urgently, they need to be planned in the next sprint, while others that are not in the urgent scope can be put into the backlog after comparing the priority with other requirements.

Data collection (due to data sensitivity, all data in the pictures have been specially manipulated)

Collect data for the next design iteration

This is the place to think about measurable data goals, what are we collecting this data for? What kind of questions do we need to answer? Is it a goal for our next iteration or future design? How is it going to be collected? In what visual way does it need to be presented?

Iterative direction exploration

Based on market demand, user needs and business goals, the designer needed to think and explore different directions for the next iteration and confirm with stakeholders and users to be fully prepared for the next iteration direction. Based on the communication with stakeholders, user research and market research, I thought of the following three iterations:

  • It serves as a sales place for rendering materials and other model-related materials.
  • Users can quickly generate their own personal portfolio websites through the website, which provides templates to help users generate them with one click.
  • Become a platform for recruitment as well as job hunting.
